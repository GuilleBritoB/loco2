Git y Github

Git

  • Es un software
  • Es mantenido por linux
  • Es un repositorio LOCAL
  • Es de codigo abierto
  • No tiene gestion de usuarios
  • Se basa en sistema de comandos
  • Control de versiones

Github

  • Es un servicio
  • Es mantenido por microsoft
  • Es un repositorio remoto
  • Es gratuito y pago
  • tiene incorporado la gestion de usuarios
  • Almacena versiones repositorios de git
